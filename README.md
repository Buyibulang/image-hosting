# Image Hosting for WeChat

This repository is used for hosting images for WeChat Official Account articles.

## Image URLs

Images can be accessed via:

```
https://raw.githubusercontent.com/Buyibulang/image-hosting/main/images/filename.jpg
```

## How to Use

1. Upload images to the `images/` directory
2. Get the raw URL for the image
3. Use the URL in your WeChat articles

## Note

GitHub Pages is not required for this workflow. We use raw.githubusercontent.com URLs which work perfectly in WeChat articles.
